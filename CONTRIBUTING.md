# Contributing to USGinger Longevity Studies

Thank you for your interest in helping us advance agricultural stasis technology! We welcome researchers, data scientists, and developers to help analyze our historical longevity datasets.

As this repository contains proprietary data protected under the **USGinger Restrictive Non-Commercial Research License**, please review these guidelines before contributing.

## How You Can Help
We are currently seeking contributions in the following areas:
* **Data Visualization:** Creating Python/R scripts to plot metabolic trends from our COA data.
* **Statistical Analysis:** Verifying standard deviation and significance metrics across the 9-year study.
* **Documentation:** Improving clarity, fixing typos, or translating research files.

## Contribution Process

### 1. Report Issues & Ask Questions
* Use our **Issue Templates** to submit questions about the study data or report discrepancies.
* Do not submit sensitive proprietary feedback publicly; email us directly for commercial inquiries.

### 2. Submitting Changes (Pull Requests)
1. **Fork** the repository and create your branch (`feature/analysis-improvement`).
2. Ensure any code scripts (Python, Jupyter Notebooks) are thoroughly commented.
3. Submit a **Pull Request (PR)** targeting the `main` branch.
4. Clearly describe your findings, changes, or visualizations in the PR description.

## Code of Conduct
We expect all contributors to maintain a respectful, professional, and scientifically rigorous environment in all issues and discussion threads.
